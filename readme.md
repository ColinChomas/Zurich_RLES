# Running


## things to do

`npm install`

If not exists => Create the .gitignore file 
- Add node_modules and .env

Create .env in root dir
```
PGHOST=localhost
PGDATABASE= What Ever your DB is
PGUSER= What Ever Your Postgres User is
PGPASSWORD= What Ever Your Password Is
PGPORT=5432
```
Add these things to the .env, this will allow you to connect locally...
